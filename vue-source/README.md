# ArticleWebApp

Vue 文章网页查看，主要用于博客园 Android 端展示文章内容、文章评论。

## Project setup

```
yarn install
```

### Compiles and hot-reloads for development

```
yarn run serve
```

### Compiles and minifies for production

```
yarn run build
```

### Run your tests

```
yarn run test
```

### Lints and fixes files

```
yarn run lint
```
